# Gamified Alpha Quest 🎮

> 🤖 **Built with AI-native workflow** — Lovable AI (primary builder) + Cursor
> Developed using Vibe Coding methodology: product spec → AI-generated code → iterative refinement.

## Overview

A gamified crypto alpha discovery platform where users compete to find and share high-signal investment insights. Combines Web3 data feeds, gamification mechanics, and AI-curated content to make on-chain research engaging.

## 🛠 Tech Stack

- **Frontend**: React, TypeScript, Tailwind CSS, shadcn-ui, Vite
- **AI Tooling**: Lovable AI, Cursor
- **Web3**: On-chain data interactions, crypto market feeds

## ✨ Features

- Gamified alpha discovery with points and leaderboards
- Crypto signal aggregation and scoring
- User quest system for on-chain research tasks
- Real-time market data integration

## 🚀 Getting Started

```bash
npm install
npm run dev
```
